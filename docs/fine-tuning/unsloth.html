Unsloth

How Unsloth speeds up QLoRA fine-tuning by 2× with less VRAM. Installation, supported models, and a complete training script.

What Is Unsloth?

Unsloth is a drop-in replacement for HuggingFace's training stack that is 2–5× faster and uses 50–70% less VRAM. It achieves this through hand-written CUDA kernels for attention, custom backprop for LoRA, and Flash Attention 2 integration. Same results, half the memory, twice the speed.

MetricStandard HuggingFaceUnsloth
Training speedbaseline2–5× faster
VRAM usagebaseline40–70% less
Qualityreferenceidentical
Min VRAM for 7B QLoRA~16 GB~8 GB ✅

Installation

bash
# Google Colab (T4 GPU — recommended starting point)
pip install "unsloth[colab-new] @ git+https://github.com/unslothai/unsloth.git"
pip install --no-deps trl peft accelerate bitsandbytes

# Local NVIDIA GPU
pip install "unsloth[cu121-ampere-torch230] @ git+https://github.com/unslothai/unsloth.git"

# Verify install
python -c "from unsloth import FastLanguageModel; print('Unsloth OK')"

Complete Training Script

python
from unsloth import FastLanguageModel
from trl import SFTTrainer
from transformers import TrainingArguments
from datasets import load_dataset

# 1. Load model in 4-bit with Unsloth
model, tokenizer = FastLanguageModel.from_pretrained(
    model_name="unsloth/Llama-3.2-1B-Instruct",
    max_seq_length=2048,
    load_in_4bit=True,    # QLoRA
)

# 2. Add LoRA adapters
model = FastLanguageModel.get_peft_model(
    model, r=16, lora_alpha=32,
    target_modules=["q_proj","k_proj","v_proj","o_proj","gate_proj","up_proj","down_proj"],
    use_gradient_checkpointing="unsloth",  # saves extra VRAM
)

# 3. Load dataset and train
dataset = load_dataset("your-username/your-dataset", split="train")

trainer = SFTTrainer(
    model=model, tokenizer=tokenizer,
    train_dataset=dataset,
    dataset_text_field="text",
    max_seq_length=2048,
    args=TrainingArguments(
        per_device_train_batch_size=2,
        gradient_accumulation_steps=4,
        learning_rate=2e-4,
        num_train_epochs=3,
        fp16=True,
        output_dir="./output",
        logging_steps=10,
    ),
)
trainer.train()

# 4. Save and export
model.save_pretrained("my-model-lora")
model.save_pretrained_gguf("my-model-gguf", tokenizer, quantization_method="q4_k_m")
model.push_to_hub_gguf("your-username/my-model-GGUF", tokenizer, token="hf_...")
