Recommendations by Budget

💚
$0 — Google Colab
Free T4 16GB. Fine-tunes 1B models. Limited to 4–6 hr sessions. Perfect to start.
💙
$0.20/hr — RunPod RTX 3090
24GB VRAM. Fine-tunes 7B with QLoRA. Pay only when training. Best cloud value.
🧡
$500 owned — RTX 3090
Buy used. Unlimited training hours. Pays for itself after ~2,500 hours vs RunPod.
💜
$1,600 owned — RTX 4090
Fastest consumer GPU. Same VRAM as 3090 but 2x faster training. For serious builders.

Apple Silicon (M1/M2/M3)

Apple Silicon uses unified memory — the GPU and CPU share the same RAM pool. An M3 Max with 96GB unified memory can run and fine-tune models that won't fit on any consumer NVIDIA GPU.
