RunPod Deployment

Best price for dedicated GPU servers. RTX 3090 24GB at $0.20/hr. Good for low-to-medium volume APIs.

1
Create a pod
runpod.io → Deploy → GPU Cloud → Select RTX 3090 → Choose PyTorch template → Deploy On-Demand.
2
SSH in and install vLLM
bash
pip install vllm huggingface_hub
huggingface-cli login  # paste your HF token
3
Start the API server
bash
python -m vllm.entrypoints.openai.api_server \
  --model your-username/YourFineTunedModel \
  --host 0.0.0.0 --port 8000 \
  --max-model-len 4096
4
Expose the port
RunPod → your pod → Connect → HTTP Service → Port 8000 → Expose. You get a public URL.

Modal — Serverless (Recommended)

Modal autoscales to zero when not in use. You only pay per second of GPU time. $30/mo free credit.

python
pip install modal
modal setup  # authenticate

# deploy_slm.py
import modal

app = modal.App('my-slm')
image = modal.Image.debian_slim().pip_install('vllm', 'huggingface_hub')

@app.cls(gpu='A10G', image=image)
class Model:
    @modal.enter()
    def load(self):
        from vllm import LLM
        self.llm = LLM('your-username/YourModel', max_model_len=4096)

    @modal.method()
    def generate(self, prompt: str) -> str:
        from vllm import SamplingParams
        out = self.llm.generate(prompt, SamplingParams(max_tokens=512))
        return out[0].outputs[0].text

@app.local_entrypoint()
def main():
    m = Model()
    print(m.generate.remote('Hello!'))

# Deploy
# modal deploy deploy_slm.py
