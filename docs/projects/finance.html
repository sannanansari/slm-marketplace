Build a Finance Parser SLM

Extract structured financial metrics from earnings calls, 10-K filings, and analyst reports. JSON output ready for quant pipelines.

⏱ 3 hours trainingIntermediate💰 ~$6 on RunPod

Use Case

Quantitative analysts spend hours manually extracting figures from earnings call transcripts and SEC filings. A fine-tuned SLM extracts revenue, EPS, guidance, and sentiment in milliseconds — at zero marginal cost per filing.

Free Training Data: SEC EDGAR

All SEC filings are public and free to download via the EDGAR API. Earnings call transcripts are available from Motley Fool, Seeking Alpha, and Earningswhispers.

python
import requests

def get_10k_filings(ticker, n=10):
    # SEC EDGAR full-text API - completely free
    cik_url = f'https://data.sec.gov/submissions/CIK{ticker}.json'
    filings = requests.get(cik_url).json()
    
    forms = filings['filings']['recent']
    tenk_idx = [i for i, f in enumerate(forms['form']) if f == '10-K'][:n]
    
    results = []
    for i in tenk_idx:
        acc = forms['accessionNumber'][i].replace('-', '')
        doc = forms['primaryDocument'][i]
        url = f'https://www.sec.gov/Archives/edgar/data/{filings["cik"]}/{acc}/{doc}'
        results.append({'url': url, 'date': forms['filingDate'][i]})
    return results

filings = get_10k_filings('AAPL')
print(f'Found {len(filings)} 10-K filings for AAPL')

Best Base Model for Finance

Qwen2.5-3B is the recommended base model for financial parsing. It has the strongest numerical reasoning at small sizes, native JSON output capability, and excellent performance on structured extraction tasks. Qwen2.5 models are Apache 2.0 licensed — safe for commercial use.

python
from unsloth import FastLanguageModel

# Qwen2.5-3B: best for structured numerical extraction
model, tokenizer = FastLanguageModel.from_pretrained(
    model_name='unsloth/Qwen2.5-3B-Instruct',
    max_seq_length=2048,
    load_in_4bit=True,
)

# System prompt: instruct for JSON output
SYSTEM = (
    'You are a financial analyst. Extract key metrics from earnings reports.'
    'Always respond in valid JSON matching the provided schema.'
)
