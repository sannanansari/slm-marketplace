Project Overview

Medical coding is one of the highest-ROI SLM applications. US hospitals process millions of patient encounters per year, each requiring ICD-10-CM diagnosis codes and CPT procedure codes for billing. Automated coding reduces a 92-minute manual task to seconds.

Research basis: Indiana University 2025 study in npj Health Systems showed fine-tuned Llama-3.2-1B achieves 69.20% exact match on real clinical notes — vs ~34% for raw GPT-4.

⚠️
Medical disclaimer

This model is an assistive tool for trained medical coders. It should not be used for billing without human verification by a certified coder.

Dataset

Option A — MIMIC-IV (Recommended): Real clinical notes with expert-assigned ICD codes. Free for researchers. Apply at physionet.org (1-week approval).

Option B — Synthetic: Generate 5,000 realistic clinical note examples using Claude API. ~$8 cost. No access required. This is what the training code uses by default.

bash
# Download the complete training package from SLM Marketplace
# It includes: train.py, build_dataset.py, evaluate.py, inference.py

# Install dependencies
pip install unsloth transformers datasets peft trl bitsandbytes

# Build dataset (synthetic, no MIMIC access needed)
python build_dataset.py
# Creates: data/medcoder_train.jsonl (4,000 examples)
#          data/medcoder_eval.jsonl  (1,000 examples)

Training

bash
# Run on Colab T4 (~4 hours) or RunPod RTX 3090 (~1.5 hours)
python train.py

# Key hyperparameters (from train.py):
# base_model   = meta-llama/Llama-3.2-1B-Instruct
# lora_r       = 16
# lora_alpha   = 32
# epochs       = 3
# batch_size   = 4 (effective 16 with grad accum)
# learning_rate= 2e-4
# max_seq_len  = 1024

Deployment

python
from inference import MedCoder

mc = MedCoder('your-username/MedCoder-SLM-1B')

note = """
72-year-old male with hypertension and T2DM presented with acute chest pain,
ST elevation V1-V4. Troponin 2.8. LAD stent placed emergently.
"""

result = mc.code(note)
print(result)
