Build a Coding Assistant SLM

Local code completion and review at sub-100ms. Your code never leaves your machine.

⏱ 5 hours trainingIntermediate

Why a Local Coding SLM?

GitHub Copilot sends your code to the cloud — a problem for proprietary codebases, financial systems, and healthcare applications. A local 1B-3B coding SLM running on your GPU or Apple Silicon delivers:

  • Sub-100ms completion — faster than Copilot's API round trip
  • Zero data leakage — no code leaves your machine
  • Custom training — trained on your own codebase conventions
  • Free after training — no $19/month subscription

Training Dataset

Combine multiple free sources:

  • The Stack v2 (HuggingFace) — 619 programming languages, permissive licenses only
  • CodeSearchNet — function-docstring pairs in 6 languages. Perfect for docstring generation.
  • CVE-fixing commits — GitHub commits that fix security vulnerabilities. For security review training.
  • Your own codebase — Most valuable for domain-specific completion. Format as file-completion pairs.
python
# Format for code completion training
def format_completion_example(prefix, completion):
    return {
        'instruction': 'Complete this code:',
        'input': prefix,
        'output': completion
    }

# Format for code review training
def format_review_example(code, issues):
    return {
        'instruction': 'Review this code for bugs and security issues:',
        'input': code,
        'output': issues  # e.g. 'SQL injection vulnerability on line 12: ...'
    }

Serve in VS Code via Ollama

bash
# 1. Convert to GGUF and load into Ollama
ollama create mycodeslm -f Modelfile

# 2. Install Continue extension in VS Code
# marketplace.visualstudio.com/items?itemName=Continue.continue

# 3. Configure Continue to use your local model
