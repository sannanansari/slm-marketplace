Popular Small Language Models in 2025

The best SLMs available right now: Llama 3.2, Phi-3.5, Gemma 2, Qwen 2.5, Mistral 7B — benchmarks, licenses, and when to use each.

How to Choose a Base Model

Pick your base model before building a dataset. The base model determines your ceiling. For most fine-tuning tasks: Phi-3.5-mini (best reasoning per parameter) or Llama 3.2-3B (best ecosystem support) are the defaults in 2025.

Quick Decision Guide

Commodity CPU / laptop (no GPU): Llama 3.2-1B Q4_K_M via Ollama
Consumer GPU (RTX 3080/4080): Phi-3.5-mini or Llama 3.2-3B
Commercial use, need MIT license: Phi-3.5-mini or SmolLM2
Multilingual: Qwen2.5
Code generation: Mistral-7B or Qwen2.5-Coder
