Google Colab Setup

Free T4 GPU, runs in browser, zero local install. The fastest way to start fine-tuning SLMs.

⏱ 5 min setupBeginner💰 Free

Get Started in 5 Steps

1
Go to colab.google.com
Sign in with your Google account. Click New notebook.
2
Enable GPU runtime
Runtime → Change runtime type → Hardware accelerator → T4 GPU → Save.
3
Verify GPU
python
!nvidia-smi
import torch
print(torch.cuda.get_device_name(0))
print(f'VRAM: {torch.cuda.get_device_properties(0).total_memory/1e9:.1f} GB')
4
Install Unsloth (fastest SLM training)
bash
%%capture
!pip install unsloth
!pip install --no-deps trl peft accelerate bitsandbytes
5
Load a model and start training
python
from unsloth import FastLanguageModel

model, tokenizer = FastLanguageModel.from_pretrained(
    model_name='unsloth/Llama-3.2-1B-Instruct',
    max_seq_length=2048,
    load_in_4bit=True,  # QLoRA
)
print('Model loaded successfully!')

⚠️
Save your work!

Colab sessions are temporary. Always save your trained model to Google Drive or push to HuggingFace Hub before the session ends.

Tips to Maximize Free GPU Hours

  • Don't open multiple Colab tabs — they share your GPU quota
  • Use Runtime → Disconnect and delete runtime when done to free your quota
  • Save checkpoints to Google Drive every 30 minutes during long training runs
  • Use Unsloth — it's 2x faster and uses 70% less VRAM, so training finishes before timeout
  • For runs over 4 hours, use RunPod or Kaggle instead
