Quick Start

🚀
No GPU? Start with Google Colab

Google Colab gives you a free T4 GPU (16GB VRAM). Runs fine-tuning for Llama 3.2-1B. No setup, runs in browser, free. Colab Setup Guide →

Local Setup

🪟
Windows
WSL2 + CUDA + Python + PyTorch. Full guide with commands.
🍎
macOS
Apple Silicon (M1/M2/M3) with Metal backend. Homebrew setup.
🐧
Linux
Ubuntu 22.04/24.04 with NVIDIA drivers and CUDA toolkit.
