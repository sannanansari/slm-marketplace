RunPod

How to rent a GPU on RunPod for SLM fine-tuning. Setup, cost, templates, and connecting VS Code to your pod.

Why RunPod?

Google Colab free tier gives you a T4 (16GB VRAM) but disconnects after 90 minutes. RunPod rents GPUs by the hour — an RTX 4090 (24GB) costs ~$0.44/hr, an A100 (80GB) ~$2.50/hr. For serious training runs (2+ hours), RunPod is more reliable than Colab Pro.

Getting Started

bash
# 1. Create account at runpod.io # 2. Add credits ($10 minimum) # 3. Go to Pods → Deploy → select GPU # 4. Use template: "RunPod PyTorch 2.1" (pre-installed CUDA) # 5. Deploy and connect via SSH or JupyterLab # Once connected to your pod: pip install unsloth transformers datasets trl peft # Clone your training script git clone https://github.com/your-username/your-training-repo cd your-training-repo python finetune.py # Monitor GPU watch -n1 nvidia-smi # Save model to network volume BEFORE terminating the pod! # Data on the pod itself is lost when terminated.
Critical: Always save your model to a Network Volume or push to HuggingFace Hub before terminating a pod. On-disk data is lost when pods are terminated.

Cost Estimate for Training

Fine-tuning Llama 3.2-1B on 500 examples with Unsloth (RTX 4090, 3 epochs): approximately 25 minutes = ~$0.18 total. That's less than a coffee.
