🍎 Setup

macOS Setup for SLM Development

Set up your Mac (M1/M2/M3/M4 or Intel) for SLM development. Python, PyTorch with Metal, Ollama, and HuggingFace tools.

Mac for AI Development

Apple Silicon Macs (M1/M2/M3/M4) have unified memory — the GPU and CPU share RAM. A 32GB M3 Mac can run a 7B model at 13GB directly in memory using Metal acceleration. Excellent for local inference; limited for training large models.

Complete Setup

bash
# 1. Install Homebrew /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" # 2. Python 3.11 brew install python@3.11 python3.11 --version # 3. Virtual environment python3.11 -m venv ~/slm-env source ~/slm-env/bin/activate echo "source ~/slm-env/bin/activate" >> ~/.zshrc # 4. PyTorch with Metal (Apple Silicon) pip install torch torchvision torchaudio # Metal backend is auto-detected # Verify Metal GPU python -c " import torch print('MPS available:', torch.backends.mps.is_available()) print('Device:', torch.device('mps')) " # 5. HuggingFace stack pip install transformers datasets accelerate peft trl pip install huggingface_hub sentence-transformers # 6. Ollama (native Apple Silicon support, uses Metal automatically) brew install ollama ollama serve & ollama run llama3.2:1b

Training on Mac

Training limitations on Mac: bitsandbytes (required for QLoRA) has limited Mac support. Use unsloth on Mac for fine-tuning, or use Google Colab (free T4 GPU) for training and Mac for inference only. Mac is excellent for inference — Ollama runs at full Metal speed.
