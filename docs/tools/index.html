Every tool you need to build, train, evaluate, and deploy Small Language Models — curated and categorised.

Training & Fine-Tuning

🦥
Unsloth
2x faster fine-tuning, 70% less VRAM than standard HuggingFace. Works with Llama, Phi, Mistral, Qwen. Free and open-source.
Open SourceFreeMost Popular 2026
🤗
Hugging Face PEFT
The standard library for LoRA, QLoRA, and adapter-based fine-tuning. Integrates with Transformers and TRL.
Open SourceFreeIndustry Standard
🎯
TRL (Transformer Reinforcement Learning)
HuggingFace's library for SFT, DPO, PPO, and RLHF training. SFTTrainer is the go-to for instruction tuning.
Open SourceFree
⚡
Axolotl
Configuration-driven fine-tuning. Write a YAML file, run one command. Supports LoRA, QLoRA, full fine-tuning, RLHF.
Open SourceFree

Inference Engines

🦙
Ollama
Easiest way to run SLMs locally. CLI + REST API. Supports Mac, Windows, Linux. 300+ models available via pull command.
Open SourceFreeMost Popular Local
⚡
vLLM
Highest throughput GPU serving. PagedAttention delivers 10–25x more requests/second vs naive inference. OpenAI-compatible API.
Open SourceFreeBest GPU Server
🔧
llama.cpp
CPU inference for GGUF models. Runs on any machine without a GPU. Foundation for Ollama and many other tools.
Open SourceFree

Evaluation

🏆
LM Evaluation Harness (EleutherAI)
Industry standard. 200+ benchmark tasks. ARC, HellaSwag, MMLU, TruthfulQA. Run with one command.
Open SourceFreeIndustry Standard
📊
Weights & Biases (W&B)
Experiment tracking, metric logging, model comparison dashboards. Essential for tracking training runs. Free for individuals.
Free tierPopular

Dataset Tools

📦
Hugging Face Datasets
100,000+ public datasets. Load in one line. Supports streaming for large datasets. Arrow-based for fast processing.
Open SourceFreeIndustry Standard
🏷️
Label Studio
Open-source data labeling. Text, image, audio. Self-host for free or use cloud version for team labeling.
Open SourceFree self-hosted
