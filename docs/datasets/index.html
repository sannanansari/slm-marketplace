Dataset Academy

Your model is only as good as your data. Learn to build high-quality training datasets from scratch.

⏱ 15 min readIntermediate

Dataset Format

Most SLM fine-tuning uses the instruction-following format — pairs of (instruction, input, output):

json
[
  {
    "instruction": "Classify this note and assign ICD-10 codes.",
    "input": "Patient with acute chest pain, ST elevation V1-V4.",
    "output": "I21.9 - Acute myocardial infarction, unspecified"
  },
  {
    "instruction": "Extract the limitation of liability clause.",
    "input": "...contract text...",
    "output": "Section 8.2: Liability shall not exceed fees paid in 12 months."
  }
]

Data Collection Strategies

🌐
Public Datasets
HuggingFace Hub has 100,000+ datasets. CUAD (legal), MIMIC-IV (medical), CodeSearchNet (code). Filter by license.
🏢
Internal Data
Your company's documents, tickets, emails. Highest-quality signal for domain tasks. Ensure PII is removed.
🤖
Synthetic Generation
Use GPT-4 or Claude to generate training examples from templates. Fast, scalable, controllable quality.
🔍
Web Scraping
Scrapy, BeautifulSoup, Common Crawl. Always check robots.txt and terms of service before scraping.

Data Cleaning Checklist

Bad data is worse than less data. Run these checks before training:

  • Remove examples where output is empty or under 10 characters
  • Deduplicate — identical or near-identical examples hurt training
  • Check for PII — names, emails, phone numbers in training data
  • Normalize whitespace and encoding
  • Truncate outliers — outputs over 512 tokens slow training significantly
  • Spot-check 50 random examples manually — quality issues are often obvious
python
from datasets import load_dataset
import re

dataset = load_dataset("json", data_files="my_data.jsonl")

def clean(ex):
    if not ex["output"] or len(ex["output"].strip()) < 10:
        return None
    ex["output"] = re.sub(r"\s+", " ", ex["output"]).strip()
    if len(ex["output"].split()) > 512:
        ex["output"] = " ".join(ex["output"].split()[:512])
    return ex

cleaned = dataset.filter(lambda x: clean(x) is not None)
print(f"Kept {len(cleaned['train'])} / {len(dataset['train'])} examples")

Synthetic Data Generation

When real data is scarce, use a larger model to generate examples:

python
import anthropic, json

client = anthropic.Anthropic()

def generate_examples(n=100):
    examples = []
    for _ in range(n):
        msg = client.messages.create(
            model="claude-sonnet-4-6", max_tokens=500,
            messages=[{"role": "user", "content":
                "Generate a clinical note and its ICD-10 codes.\n"
                'Return JSON: {"note": "...", "codes": ["X00.0"]}'
            }]
        )
        try:
            data = json.loads(msg.content[0].text)
            examples.append({
                "instruction": "Assign ICD-10 codes.",
                "input": data["note"],
                "output": ", ".join(data["codes"])
            })
        except: pass
    return examples
